# Loop-Practice 🔄

This repository is dedicated to mastering **Loops and Iterations** in Java. This project covers the fundamentals of iterative logic, ranging from simple counters to sequential logic and table generation.

---

## 🚀 Projects Included

### For Loops
* **`For_Loop_Practice.java`:** Simple iteration to understand counting logic by printing numbers from 1 to 10.
* **`For_Loop_Sum_Practice.java`:** Dynamic range checker using accumulation logic to find the sum of first `n` natural numbers.
* **`Table_Generator_Loop.java`:** Multiplication table generator for a user-specified input number.

### While Loops
* **`While_Loop_Practice.java`:** Sequential counter logic demonstrating entry-controlled loop execution from 1 to 10.

### Do-While Loops
* **`Do_While_Loop_Practice.java`:** Exit-controlled loop mechanism guaranteeing at least one execution block while counting.

---

## 🛠️ Concepts Covered

* **Iterative Logic:** Implementing `for`, `while`, and `do-while` constructs cleanly.
* **Input Handling:** Using `java.util.Scanner` for dynamic, user-interactive programs.
* **Defensive Programming:** Ensuring memory safety with `sc.close()` and robust condition management.
* **Scope Resolution:** Understanding block-level variable initialization inside loops.
